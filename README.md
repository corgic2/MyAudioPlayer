# MyAudioPlayer
音视频播放器

## 项目概述
这是一个基于C++和Qt框架开发的音视频播放器项目，包含完整的媒体播放功能和用户界面。

## 项目模块
### 1. 核心播放器 (MyAudioPlayer)
- **音视频解码**：基于FFmpeg的多格式媒体文件解码
- **音频处理**：音频重采样、格式转换、波形显示
- **视频处理**：视频解码、帧渲染、播放控制  
- **用户界面**：Qt界面组件，包括播放控制、进度条、音量控制

### 2. 自定义UI组件 (MyCustomUIWidget)
- **核心组件**：按钮、进度条、标签等基础UI组件
- **域组件**：文件列表、拖拽组件等专用组件
- **Qt设计器插件**：可在Qt Designer中使用的自定义组件

### 3. 通用SDK (MySdk)
- **数据库系统**：SQLite数据库管理和查询构建器
- **文件系统**：文件操作和路径管理
- **日志系统**：多级别日志记录和输出
- **线程池**：高效的任务调度和线程管理
- **时间系统**：时间处理和格式化工具

## 使用说明

### 音视频播放器使用
1. 编译项目：运行 `BuildVSProject_Release.bat`
2. 运行程序：启动生成的可执行文件
3. 加载媒体文件：支持多种音视频格式
4. 播放控制：播放、暂停、快进、音量调节

### 自定义UI组件使用
1. 编译组件库：运行对应的构建脚本
2. 安装设计器插件：将插件加载到Qt Designer
3. 使用组件：在Qt Designer中拖拽使用自定义组件

### SDK使用
1. 包含头文件：`#include "MySdk/xxx.h"`
2. 链接库文件：添加相应的库文件
3. 调用API：参考各模块的使用示例

2. 实践练习：每个概念配有完整的代码示例
3. 参考查询：可作为C语言开发的参考手册
